## 任务1：图的基本概念（60分钟）

**图的定义**：
- 图G = (V, E)
- V：顶点集合
- E：边集合

**图的分类**：

| 类型 | 特点 | 示例 |
|------|------|------|
| 无向图 | 边无方向 | 道路网络 |
| 有向图 | 边有方向 | 网页链接 |
| 无权图 | 边无权重 | 简单关系 |
| 有权图 | 边有权重 | 路程距离 |

**图的基本术语**：

| 术语 | 定义 |
|------|------|
| 度 | 顶点关联的边数 |
| 入度 | 有向图中进入该顶点的边数 |
| 出度 | 有向图中从该顶点出发的边数 |
| 路径 | 从一个顶点到另一个顶点的边序列 |
| 连通图 | 任意两顶点都有路径相连 |
| 强连通图 | 有向图中任意两顶点双向连通 |
| 完全图 | 每两个顶点之间都有边 |

**完全图的边数**：
- 无向完全图：n(n-1)/2 条边
- 有向完全图：n(n-1) 条边

## 任务2：图的存储方式（30分钟）⭐重点

**邻接矩阵**：

| 特性 | 说明 |
|------|------|
| 存储方式 | 二维数组 |
| 空间复杂度 | O(n²) |
| 适用场景 | 稠密图、频繁查询边 |
| 判断边存在 | O(1) |
| 查找所有邻接点 | O(n) |

**示例**：
```
无向图：
  A-B, A-C, B-C

邻接矩阵：
    A  B  C
A [ 0  1  1 ]
B [ 1  0  1 ]
C [ 1  1  0 ]
```

**邻接表**：

| 特性 | 说明 |
|------|------|
| 存储方式 | 链表数组 |
| 空间复杂度 | O(n+e) |
| 适用场景 | 稀疏图、频繁遍历 |
| 判断边存在 | O(d)（d为度） |
| 查找所有邻接点 | O(d) |

**示例**：
```
A → [B, C]
B → [A, C]
C → [A, B]
```

## 任务3：练习题目（30分钟）

1. 5个顶点的无向完全图有多少条边？
   - 答案：n(n-1)/2 = 5×4/2 = 10

2. 有向图中，所有顶点入度之和等于？
   - 答案：等于出度之和，等于边数

3. 稀疏图用什么存储更节省空间？
   - 答案：邻接表

4. 需要频繁判断两顶点是否有边，用什么存储？
   - 答案：邻接矩阵
