## 一、四种遍历方式（20题）

### 示例二叉树1
```
      A
     / \
    B   C
   / \   \
  D   E   F
```

### 第1题
写出上述二叉树的前序遍历结果。

**答案：A B D E C F**

**解析**：
前序遍历：根 → 左 → 右
A → (B → D → E) → (C → F)

---

### 第2题
写出上述二叉树的中序遍历结果。

**答案：D B E A C F**

**解析**：
中序遍历：左 → 根 → 右
(D → B → E) → A → (C → F)

---

### 第3题
写出上述二叉树的后序遍历结果。

**答案：D E B F C A**

**解析**：
后序遍历：左 → 右 → 根
(D → E → B) → (F → C) → A

---

### 第4题
写出上述二叉树的层序遍历结果。

**答案：A B C D E F**

**解析**：
层序遍历：从上到下，从左到右
第1层：A
第2层：B C
第3层：D E F

## 二、遍历序列还原二叉树 ⭐重点必考（10题）

### 第21题 ⭐重点
给定前序ABDECF，中序DBEACF，画出二叉树并写出后序。

**答案：后序 = DEBFCA**

**解析**：
1. 前序第一个A是根
2. 中序A左边DBE是左子树，右边CF是右子树
3. 前序对应左子树部分：BDE（B是左子树根）
4. 中序DBE：D在B左边，E在B右边
5. 前序对应右子树部分：CF（C是右子树根）
6. 中序CF：只有F在C右边

```
      A
     / \
    B   C
   / \   \
  D   E   F
```

---

### 第22题 ⭐重点
给定前序ABCDE，中序BADCE，画出二叉树并写出后序。

**答案：后序 = BDECA**

**解析**：
1. A是根
2. 中序B在A左边 → B是左孩子（无子树）
3. 中序DCE在A右边 → C是右子树根
4. D在C左边，E在C右边

```
      A
     / \
    B   C
       / \
      D   E
```

---

### 第23题
给定前序ABDC，中序DBAC，画出二叉树并写出后序。

**答案：后序 = DBCA**

**解析**：
1. A是根
2. 中序DB在A左边 → B是左子树根，D是B的左孩子
3. C在A右边 → C是右孩子

```
      A
     / \
    B   C
   /
  D
```

---

### 第24题
给定前序ABC，中序BAC，画出二叉树并写出后序。

**解析**：

```
    A
   / \
  B   C
```

---

### 第25题

### 第26题 ⭐重点
前序+中序能否唯一确定二叉树？

**答案：能**

**解析**：
前序找根，中序分左右，递归可唯一确定

---

### 第27题 ⭐重点
后序+中序能否唯一确定二叉树？

**答案：能**

**解析**：
后序找根（最后一个），中序分左右，递归可唯一确定

---

### 第28题 ⭐重点
前序+后序能否唯一确定二叉树？

**答案：不能**

**解析**：
缺少中序信息，无法区分左右子树
例：前序AB，后序BA，可能是B左孩子或B右孩子

---

### 第29题
为什么前序+后序不能唯一确定二叉树？

**解析**：
前序找根，后序找根（都是根）
但没有中序，无法知道左子树和右子树的边界
如前序AB，后序BA：
- 可能是A(B,)，B是左孩子
- 可能是A(,B)，B是右孩子
两种结构的前序后序相同

---

### 第30题
要唯一确定二叉树，必须有什么遍历序列？

**答案：中序遍历**

**解析**：
中序遍历能区分左右子树的边界
前序+中序 或 后序+中序 都能唯一确定

## 四种遍历总结 ⭐必背

| 遍历方式 | 顺序 | 助记 |
|----------|------|------|
| 前序 | 根-左-右 | 根在最前 |
| 中序 | 左-根-右 | 根在中间 |
| 后序 | 左-右-根 | 根在最后 |
| 层序 | 上到下左到右 | 用队列 |

---

## 还原二叉树关键结论 ⭐必考

| 组合 | 能否唯一确定 |
|------|--------------|
| 前序+中序 | ✅ 能 |
| 后序+中序 | ✅ 能 |
| 前序+后序 | ❌ 不能 |

**原因**：必须有中序才能区分左右子树边界
