## 学习任务清单

### 任务1：四种遍历方式（30分钟）⭐重点

**遍历方式定义**：

| 遍历方式 | 访问顺序 | 说明 |
|----------|----------|------|
| **前序遍历** | 根 → 左 → 右 | 先访问根 |
| **中序遍历** | 左 → 根 → 右 | 中间访问根 |
| **后序遍历** | 左 → 右 → 根 | 最后访问根 |
| **层序遍历** | 从上到下，从左到右 | 按层次遍历 |

**示例二叉树**：
```
      A
     / \
    B   C
   / \   \
  D   E   F
```

| 遍历方式 | 结果 |
|----------|------|
| 前序 | A B D E C F |
| 中序 | D B E A C F |
| 后序 | D E B F C A |
| 层序 | A B C D E F |

---

### 任务2：由遍历序列还原二叉树（20分钟）⭐必考

**关键结论**：
- **前序 + 中序** → 可唯一确定二叉树
- **后序 + 中序** → 可唯一确定二叉树
- **前序 + 后序** → 不能唯一确定（缺少中序信息）

**还原方法**：

1. 前序找根（第一个就是根）
2. 中序找左右子树（根左边是左子树，右边是右子树）
3. 递归处理左右子树

**示例**：
- 前序：A B D E C F
- 中序：D B E A C F

```
前序第一个A是根
中序A左边 D B E 是左子树，右边 C F 是右子树
继续递归...
```

### 任务3：遍历应用场景（10分钟）

| 遍历方式 | 应用场景 |
|----------|----------|
| 前序 | 复制二叉树、计算表达式树前缀表达式 |
| 中序 | 二叉排序树输出有序序列、表达式树中缀表达式 |
| 后序 | 计算表达式树值、删除二叉树、求目录大小 |
| 层序 | 按层次输出、计算树高度 |

## 练习题目

1. 给定前序ABDECF，中序DBEACF，画出二叉树
   ```
       A
      / \
     B   C
    / \   \
   D   E   F
   ```

2. 上题的后序遍历结果是？
   - 答案：DEBFCA

3. 前序ABC，后序CBA，能唯一确定二叉树吗？
   - 答案：不能，缺少中序

---

## 易错点提醒

1. 还原二叉树必须要有**中序遍历**
2. 前序找根，中序分左右
3. 层序遍历用**队列**实现
