## 内聚与耦合对比表 ⭐必背

### 内聚（模块内部）

| 类型 | 强度 | 说明 |
|------|------|------|
| 功能内聚 | 最强 | 完成单一功能 ⭐最好 |
| 顺序内聚 | 较强 | 顺序执行 |
| 通信内聚 | 中等 | 共享数据 |
| 逻辑内聚 | 较弱 | 逻辑相关 |
| 偶然内聚 | 最弱 | 无关系 ⭐最差 |

### 耦合（模块之间）

| 类型 | 强度 | 说明 |
|------|------|------|
| 内容耦合 | 最强 | 直接访问内部 ⭐最差 |
| 公共耦合 | 较强 | 共享全局数据 |
| 外部耦合 | 中等 | 共享外部资源 |
| 数据耦合 | 最弱 | 只传递数据 ⭐最好 |

---

## 关键结论 ⭐必考

| 问题 | 答案 |
|------|------|
| 设计目标 | 高内聚、低耦合 |
| 最好内聚 | 功能内聚 |
| 最差内聚 | 偶然内聚 |
| 最好耦合 | 数据耦合 |
| 最差耦合 | 内容耦合 |
