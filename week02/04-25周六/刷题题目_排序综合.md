## Week2核心知识点总结 ⭐必背

### 排序稳定性（最重要！）

| 算法 | 稳定性 |
|------|--------|
| 直接插入 | ✅稳定 |
| 冒泡排序 | ✅稳定 |
| 简单选择 | ❌不稳定 |
| 快速排序 | ❌不稳定 |
| 归并排序 | ✅稳定 |
| 堆排序 | ❌不稳定 |

**口诀：快选堆希不稳定**

---

### 时间复杂度

| 算法 | 时间复杂度 |
|------|------------|
| O(n²)排序 | 插入、冒泡、选择 |
| O(n log n)排序 | 快排、归并、堆排序 |

---

### 空间复杂度

| 算法 | 空间复杂度 |
|------|------------|
| O(1) | 插入、冒泡、选择、堆排序 |
| O(n) | 归并排序 |
| O(log n) | 快速排序 |

---

### 关键结论 ⭐必考

| 问题 | 答案 |
|------|------|
| 堆排序稳定吗 | ❌不稳定（高频陷阱！） |
| O(n log n)稳定排序 | 归并排序 |
| O(1)空间O(n log n) | 堆排序 |
| 基本有序用什么排序 | 直接插入排序 |
