## 学习任务清单

### 任务1：快速排序（20分钟）⭐重点必考

**原理**：
- 分治思想
- 选择基准，小于基准放左边，大于放右边
- 递归处理左右子序列

**算法流程**：
```
QuickSort(arr, low, high):
  if low < high:
    pivot = Partition(arr, low, high)
    QuickSort(arr, low, pivot-1)
    QuickSort(arr, pivot+1, high)

Partition(arr, low, high):
  pivot = arr[low]
  while low < high:
    while low < high and arr[high] >= pivot: high--
    arr[low] = arr[high]
    while low < high and arr[low] <= pivot: low++
    arr[high] = arr[low]
  arr[low] = pivot
  return low
```

**性能分析**：
- 时间复杂度：平均O(n log n)，最坏O(n²)
- 空间复杂度：O(log n)
- 稳定性：**不稳定**

---

### 任务2：归并排序（20分钟）

**原理**：
- 分治思想
- 分割 → 排序子序列 → 合并

**性能分析**：
- 时间复杂度：O(n log n)
- 空间复杂度：O(n) ⭐需要额外空间
- 稳定性：**稳定**

---

### 任务3：堆排序（20分钟）⭐重点必考

**原理**：
- 利用堆结构
- 大根堆：根节点最大
- 堆顶元素与末尾交换，调整堆

**堆的性质**：
- 大根堆：arr[i] >= arr[2i] 且 arr[i] >= arr[2i+1]
- 小根堆：arr[i] <= arr[2i] 且 arr[i] <= arr[2i+1]

**性能分析**：
- 时间复杂度：O(n log n)
- 空间复杂度：O(1) ⭐无额外空间
- 稳定性：**不稳定** ⭐

---

## 高级排序对比表 ⭐必背

| 算法 | 时间复杂度 | 空间复杂度 | 稳定性 | 特点 |
|------|------------|------------|--------|------|
| **快速排序** | O(n log n) | O(log n) | ❌不稳定 | 平均最快，分治 |
| **归并排序** | O(n log n) | O(n) | ✅稳定 | 需额外空间 |
| **堆排序** | O(n log n) | O(1) | ❌不稳定 | 无额外空间 |

## 练习题目

1. 时间复杂度O(n log n)且稳定的排序是？
   - 答案：归并排序

2. 时间复杂度O(n log n)且无额外空间的排序是？
   - 答案：堆排序

3. 堆排序稳定吗？
   - 答案：不稳定 ⭐

4. 快速排序最坏情况复杂度？
   - 答案：O(n²)（基准选择不当）

---

## 易错点提醒 ⭐

1. **堆排序不稳定**（考试常设陷阱）
2. **归并排序需要O(n)额外空间**
3. 快排最坏O(n²)
