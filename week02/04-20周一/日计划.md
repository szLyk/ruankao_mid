## 学习任务清单

### 任务1：深度优先搜索DFS（30分钟）

**DFS原理**：
- 沿一条路径深入到底，再回溯探索其他路径
- 使用**栈或递归**实现
- 像走迷宫：一条路走到尽头，再回来走另一条

**DFS图解演示**：
```
示例图：
    A --- B --- D
    |     |
    C --- E

从A开始DFS遍历：

Step1: 访问A，标记A已访问
    [A]--- B --- D       栈：[A]
     |     |
    C --- E

Step2: A的邻居有B、C，先选B（假设按字母顺序）
    [A]---[B]--- D       栈：[A, B]
     |     |
    C --- E

Step3: B的邻居有A（已访问）、D、E，选D
    [A]---[B]---[D]      栈：[A, B, D]
     |     |
    C --- E

Step4: D的邻居只有B（已访问），无路可走，回溯！
    [A]---[B]---[D]      栈：[A, B] ← D弹出
     |     |
    C --- E              回到B，继续找未访问邻居

Step5: B还有E未访问，访问E
    [A]---[B]---[D]      栈：[A, B, E]
     |     |
    C ---[E]

Step6: E的邻居有B（已访问）、C，访问C
    [A]---[B]---[D]      栈：[A, B, E, C]
     |     |
   [C]---[E]

Step7: C的邻居A、E都已访问，回溯...
    全部访问完毕！

DFS访问顺序：A → B → D → E → C
（一条路走到头，再回来走另一条）
```

**DFS核心思想图解**：
```
DFS = "钻洞探索"

想象你在迷宫里：
    入口A
      │
      ├─→ 路B → 路D（走到底了！）
      │         ↓
      │       回溯←←←
      │         ↓
      ├─→ 路C

先走到最深处D，发现没路了，原路返回，
回到B，发现还有路E，继续走...
```

**DFS算法流程**：
```
DFS(v):
  访问v
  标记v已访问
  for v的每个邻接点w:
    if w未访问:
      DFS(w)
```

**DFS应用**：
- 连通性检测
- 拓扑排序
- 检测环
- 路径查找

### 任务2：广度优先搜索BFS（30分钟）

**BFS原理**：
- 按层次逐层探索
- 使用**队列**实现
- 像水波纹扩散：一层一层向外扩展

**BFS图解演示**：
```
示例图（同DFS）：
    A --- B --- D
    |     |
    C --- E

从A开始BFS遍历：

Step1: 访问A，A入队
    [A]--- B --- D       队列：[A]
     |     |
    C --- E              第0层：A

Step2: A出队，A的邻居B、C入队，访问B、C
    [A]---[B]--- D       队列：[B, C]
     |     |
   [C]--- E              第1层：B, C

Step3: B出队，B的邻居D、E入队（A已访问），访问D、E
    [A]---[B]---[D]      队列：[C, D, E]
     |     |
   [C]---[E]             第2层：D, E

Step4: C出队，C的邻居A、E都已访问，无新节点
    [A]---[B]---[D]      队列：[D, E]
     |     |
   [C]---[E]

Step5: D出队，D的邻居B已访问
    [A]---[B]---[D]      队列：[E]
     |     |
   [C]---[E]

Step6: E出队，E的邻居B、C都已访问
    队列空，结束！

BFS访问顺序：A → B → C → D → E
（按层次扩散，先近后远）
```

**BFS核心思想图解**：
```
BFS = "水波纹扩散"

想象水波纹从中心向外扩散：

        第2层: D, E
       ↗         ↖
      /           \
     /  第1层: B,C  \
    ↗   ↗       ↖   ↖
   /   /         \   \
  /   /   第0层:A  \   \
 ←←←←←←←←←←←←←←←←←←←←←←
      \   \         /   /
       \   \       /   /
        ↓   ↓     ↓   ↓

先访问A（中心）
再访问B、C（第一圈）
最后访问D、E（第二圈）
```

**DFS vs BFS 过程对比图**：
```
同一个图，两种遍历方式对比：

    A --- B --- D
    |     |
    C --- E

DFS（深入）：           BFS（扩散）：
A → B → D              A（第0层）
       ↓                ↓
    回溯到B            B → C（第1层）
       ↓                ↓
    继续到E            D → E（第2层）
       ↓
    继续到C

顺序：A B D E C        顺序：A B C D E
```

**BFS算法流程**：
```
BFS(v):
  初始化队列Q
  访问v，标记已访问
  v入队Q
  while Q不为空:
    u = Q出队
    for u的每个邻接点w:
      if w未访问:
        访问w，标记已访问
        w入队Q
```

**BFS应用**：
- 最短路径（无权图）
- 层次遍历
- 连通性检测

---

## DFS vs BFS对比

| 特性 | DFS | BFS |
|------|-----|-----|
| 实现方式 | 栈/递归 | 队列 |
| 探索方式 | 深入到底再回溯 | 按层次逐层探索 |
| 空间复杂度 | O(h)（h为深度） | O(w)（w为宽度） |
| 最短路径 | 不保证 | 保证（无权图） |
| 适用场景 | 拓扑排序、路径检测 | 最短路径、层次遍历 |
