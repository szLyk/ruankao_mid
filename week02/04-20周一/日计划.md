## 学习任务清单

### 任务1：深度优先搜索DFS（30分钟）

**DFS原理**：
- 沿一条路径深入到底，再回溯探索其他路径
- 使用**栈或递归**实现

**DFS算法流程**：
```
DFS(v):
  访问v
  标记v已访问
  for v的每个邻接点w:
    if w未访问:
      DFS(w)
```

**DFS应用**：
- 连通性检测
- 拓扑排序
- 检测环
- 路径查找

---

### 任务2：广度优先搜索BFS（30分钟）

**BFS原理**：
- 按层次逐层探索
- 使用**队列**实现

**BFS算法流程**：
```
BFS(v):
  初始化队列Q
  访问v，标记已访问
  v入队Q
  while Q不为空:
    u = Q出队
    for u的每个邻接点w:
      if w未访问:
        访问w，标记已访问
        w入队Q
```

**BFS应用**：
- 最短路径（无权图）
- 层次遍历
- 连通性检测

---

## DFS vs BFS对比

| 特性 | DFS | BFS |
|------|-----|-----|
| 实现方式 | 栈/递归 | 队列 |
| 探索方式 | 深入到底再回溯 | 按层次逐层探索 |
| 空间复杂度 | O(h)（h为深度） | O(w)（w为宽度） |
| 最短路径 | 不保证 | 保证（无权图） |
| 适用场景 | 拓扑排序、路径检测 | 最短路径、层次遍历 |
