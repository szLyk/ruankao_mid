## 一、DFD图题型总结（第1题）⭐必考

### 题型1：补全外部实体

**解题技巧**：
- 外部实体用方框表示
- 找题目中的"人/系统/部门"
- 常见实体：用户、管理员、外部系统

**例题答案**：用户、银行系统、管理员

---

### 题型2：补全加工/处理

**解题技巧**：
- 加工用圆圈表示
- 找题目中的动词短语
- 常见加工：处理订单、验证信息、生成报告

**例题答案**：处理订单、验证支付、生成报表

---

### 题型3：补全数据流

**解题技巧**：
- 数据流用箭头表示
- 箭头上标注数据名称
- 从题目描述找数据名称

**例题答案**：订单信息、支付结果、报表数据

---

### 题型4：补全数据存储

**解题技巧**：
- 数据存储用双横线表示
- 找题目中的"表/文件/数据库"
- 常见存储：用户表、订单表、日志文件

**例题答案**：用户表、订单表、支付记录

---

## 二、ER图题型总结（第2题）⭐必考

### 题型1：补全实体

**解题技巧**：
- 实体用矩形表示
- 找题目中的核心对象
- 常见实体：学生、课程、订单

**例题答案**：学生、课程、教师

---

### 题型2：补全联系

**解题技巧**：
- 联系用菱形表示
- 找实体间的关系动词
- 标注联系类型（1:1, 1:N, M:N）

**例题答案**：选课（M:N）、授课（1:N）

---

### 题型3：补全属性

**解题技巧**：
- 属性用椭圆表示
- 找实体的特征描述
- 主键属性要标记

**例题答案**：学号（主键）、姓名、年龄

---

### 题型4：ER转关系模式

**解题技巧**：
- 实体→关系（属性作字段）
- 联系→关系或外键
- M:N联系单独建表

**例题答案**：
```
学生（学号，姓名，年龄）
课程（课程号，课程名）
选课（学号，课程号，成绩）
```

---

## 三、UML图题型总结（第3题）⭐必考

### 题型1：补全类名

**解题技巧**：
- 从题目描述找核心类名
- 注意继承关系（子类继承父类）

**例题答案**：Order、Customer、Product

---

### 题型2：补全属性

**解题技巧**：
- 找名词（特征描述）
- 注意类型（String, int, List等）

**例题答案**：
```
orderId: String
amount: double
customer: Customer
```

---

### 题型3：补全方法

**解题技巧**：
- 找动词（操作描述）
- 注意参数和返回类型

**例题答案**：
```
calculateTotal(): double
getCustomer(): Customer
```

---

### 题型4：补全关系 ⭐重点

**解题技巧**：
- 聚合：空心菱形（可独立）
- 组合：实心菱形（不可独立）
- 泛化：实线空心箭（继承）
- 实现：虚线空心箭（接口）

**例题答案**：
- Order - OrderItem：组合
- Customer - Order：聚合
- Employee - Manager：泛化

---

### 题型5：用例图补全

**解题技巧**：
- 参与者：人形图标（找交互的人）
- 用例：椭圆（找功能）
- include：必须包含
- extend：可选扩展

**例题答案**：
- 参与者：用户、管理员
- 用例：登录、下单、查询订单

---

## 四、Java程序设计题型（第5题）⭐推荐选做

### 题型1：补类定义

**考点**：
- `class` 关键字
- `extends` 继承
- `implements` 接口实现
- `abstract` 抽象类

**例题答案**：
```java
public class Order {
public abstract class Shape {
public class Circle extends Shape {
public class UserService implements IService {
```

---

### 题型2：补属性和方法

**考点**：
- 属性类型（private/public）
- 方法签名（参数、返回类型）
- 构造方法（与类同名）

**例题答案**：
```java
private String orderId;
private double amount;

public double calculateTotal() {
public Order(String orderId) {
```

---

### 题型3：补继承关系 ⭐重点

**考点**：
- `extends` 继承父类
- `implements` 实现接口
- `@Override` 重写注解
- `super` 调用父类方法

**例题答案**：
```java
public class Circle extends Shape {
@Override
public void draw() {
super.display();
```

---

### 题型4：补集合操作 ⭐重点

**考点**：
- `List<E>` 列表
- `Map<K,V>` 映射
- `ArrayList` 实现
- `HashMap` 实现
- `add()` 添加、`get()` 获取、`size()` 大小

**例题答案**：
```java
List<Order> orders = new ArrayList<>();
orders.add(order);
Order o = orders.get(0);

Map<String, User> userMap = new HashMap<>();
userMap.put("id", user);
User u = userMap.get("id");
```

---

### 题型5：补异常处理

**考点**：
- `try-catch` 捕获异常
- `throw` 抛出异常
- `throws` 声明异常

**例题答案**：
```java
try {
    // 可能出错的代码
} catch (Exception e) {
    // 处理异常
}
```
