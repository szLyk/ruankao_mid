## 重点知识点清单

### 数据库基础

1. **关系代数运算**

| 运算 | 符号 | 含义 |
|------|------|------|
| 选择 | σ | 按条件筛选行 |
| 投影 | π | 选择指定列 |
| 连接 | ⨝ | 按条件合并两表 |
| 自然连接 | ⨝ | 同名属性等值连接 |

2. **范式（重点！）**

| 范式 | 条件 | 解决问题 |
|------|------|----------|
| 1NF | 属性不可再分 | 消除重复组 |
| 2NF | 消非主属性对码的部分依赖 | 消除部分依赖 |
| 3NF | 消非主属性对码的传递依赖 | 消除传递依赖 |
| BCNF | 主属性不部分/传递依赖于码 | 消除所有依赖问题 |

**范式判断步骤**：
1. 找候选码（能唯一标识元组的最小属性集）
2. 找主属性（包含在任何候选码中的属性）
3. 找非主属性
4. 判断依赖关系

3. **ER图设计（下午题必考）**

- 实体：矩形
- 属性：椭圆形
- 关系：菱形
- 联系类型：1:1、1:N、M:N

**ER转关系模式**：
- 实体 → 单独的关系
- 1:1联系 → 可合并到任一实体
- 1:N联系 → 合并到N端实体
- M:N联系 → 单独建立关系，包含两端主键

### SQL语言

1. **DDL（数据定义）**
   - CREATE TABLE/VIEW/INDEX
   - ALTER TABLE
   - DROP TABLE/VIEW/INDEX

2. **DML（数据操作）**
   - INSERT INTO
   - UPDATE SET
   - DELETE FROM

3. **查询语句（重点）**
   - SELECT ... FROM ... WHERE
   - GROUP BY + HAVING（分组聚合）
   - ORDER BY（排序）
   - JOIN / 子查询

### 操作系统

1. **进程管理**

| 进程状态 | 含义 |
|----------|------|
| 运行态 | 占用CPU执行 |
| 就绪态 | 等待CPU分配 |
| 阻塞态 | 等待I/O或其他事件 |

**进程调度算法**：
- 先来先服务（FCFS）
- 短作业优先（SJF）
- 时间片轮转（RR）
- 优先级调度
- 高响应比优先（HRRN）

2. **PV操作（信号量）**

- P操作：申请资源，S = S - 1
- V操作：释放资源，S = S + 1
- 互斥：S初值 = 1
- 同步：S初值 = 0 或 N

3. **内存管理**
   - 分页：页号+页内地址
   - 分段：段号+段内地址
   - 虚拟内存：页式、段式、段页式
