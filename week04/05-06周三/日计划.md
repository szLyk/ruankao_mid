### 任务3：查询语句（30分钟）⭐重点

**基本查询**：
```sql
SELECT 列名 FROM 表名 WHERE 条件
```

**分组聚合**：
```sql
SELECT 列名, 聚合函数
FROM 表名
WHERE 条件
GROUP BY 列名
HAVING 分组条件
ORDER BY 列名
```

**聚合函数**：
- COUNT：计数
- SUM：求和
- AVG：平均值
- MAX/MIN：最大/最小

**连接查询**：
```sql
SELECT * FROM 表1 JOIN 表2 ON 连接条件
```
