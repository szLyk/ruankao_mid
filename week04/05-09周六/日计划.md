## 任务1：内存管理（60分钟）

**分页存储**：
- 页：逻辑地址单位
- 页框：物理地址单位
- 地址结构：页号 + 页内地址

**分页地址计算**：
```
逻辑地址 = 页号 × 页大小 + 页内偏移
物理地址 = 页框号 × 页大小 + 页内偏移
```

**分段存储**：
- 段：逻辑单位（如代码段、数据段）
- 地址结构：段号 + 段内地址

**虚拟内存**：
- 页式虚拟存储
- 段式虚拟存储
- 段页式虚拟存储

**页面置换算法**：
| 算法 | 特点 |
|------|------|
| OPT | 最佳置换（理论） |
| FIFO | 先进先出 |
| LRU | 最近最少使用 |
