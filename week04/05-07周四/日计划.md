## 学习任务清单

### 任务1：视图（20分钟）

**视图定义**：
- 虚表，不存储数据
- 基于查询定义

```sql
CREATE VIEW 视图名 AS SELECT语句
DROP VIEW 视图名
```

**视图特点**：
- 简化查询
- 数据安全（隐藏敏感列）
- 逻辑独立

---

### 任务2：索引（20分钟）

**索引定义**：
- 加速查询的数据结构
- 物理存储

```sql
CREATE INDEX 索引名 ON 表名(列名)
DROP INDEX 索引名
```

**索引类型**：
- 聚簇索引：数据按索引顺序存储
- 非聚簇索引：索引和数据分开存储

**索引优缺点**：
- 优点：加速查询
- 缺点：占用空间、降低更新效率
